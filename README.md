# Multitenet_buiness-automation
BotForge is a no-code chatbot builder that lets any business create a fully configured, branded AI chatbot in under 5 minutes. Answer a few questions about your business, pick a visual theme, toggle the features you need — and get a JavaScript embed snippet ready to drop into your website.

The wizard adapts its questions based on your industry (e-commerce, healthcare, SaaS, restaurant, finance, and more), so every configuration feels tailor-made rather than generic. Built as a production-grade frontend with React, TypeScript, Tailwind CSS, Framer Motion, and Zustand.
